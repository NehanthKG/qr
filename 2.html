Timer Types in PLC

Timer instructions in a programmable logic controller (PLC) are used to introduce time delays in control processes. Here are some common timer types:

  • ON Delay Timer: Delays an instruction from becoming true for a specific period after activation.
  • OFF Delay Timer: Keeps the output active for a predetermined time after the input signal turns off.
  • Retentive Timer: Maintains the accumulated time value even through power loss or changes in the rung state.

For instance, a TON Timer can be configured with the following attributes:

  • Tag: T1
  • Inputs: IN (enables the timer) and PT (sets the preset time)
  • Outputs: Q (timer output status) and ET (elapsed time)

Counter Types in PLC

Counters in programmable logic controllers (PLCs) are used to keep track of events. Here are two common counter types:

  • Up Counter: Counts up from the current value when a signal arrives at the input CU.
  • Down Counter: Decrements by one from the preset value each time the input CD turns on.

For example, a Counter can have the following attributes:

  • Tag: C1
  • Inputs: CU/CD (signal for up/down counting), R (reset the counter), and PT (preset count)
  • Outputs: Q (counter bit when PT equals CV) and CV (current value of the counter)
